# Kubernetes Intermediate

[Hub](https://alibaihaqi.github.io/learning-docs/) › Kubernetes › Intermediate

## What you'll build

You take the beginner `web` Deployment from "it runs" to "it runs well": inject
config with a ConfigMap, add liveness/readiness probes, set resource
requests/limits, roll out a new image version safely, and scale replicas up.
Everything runs on the same local kind cluster.
